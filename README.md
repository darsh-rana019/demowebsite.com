# Pragati Capitals Website

A modern fintech landing page with loan calculator, dynamic lead form, and production-ready SEO enhancements.

## Deployment

- `index.html`, `style.css`, `script.js`, and static assets are ready for static hosting.
- Place files in the root of a GitHub Pages repository or deploy as a static site to Vercel.

### GitHub Pages

1. Push the repository to GitHub.
2. Enable Pages in repository settings and set the source to the `main` branch.
3. The site will be available at `https://<your-username>.github.io/<repo-name>/`.

### Vercel

1. Connect the repository in Vercel.
2. Choose a static site deployment.
3. Vercel will deploy automatically on push.

## Notes

- Update `script.js` with your `WEBHOOK_URL` before live form submissions.
- Replace `https://yourdomain.com` in the SEO tags, `sitemap.xml`, and `robots.txt` with your actual domain.
- `favicon.svg` is included for browser support.
